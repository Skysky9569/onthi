🧪 I. Hóa học vô cơ
Chương 1: Kim loại và Phi kim
1. Kim loại (K)

Đặc điểm chung:

  • Có tính khử mạnh
  • Cho electron trong phản ứng hóa học
  • Tác dụng với axit loãng, nước, phi kim, muối

Phản ứng điển hình:

Với axit loãng: K + axit → muối + H₂↑

$$K + \text{axit} \rightarrow \text{muối} + H_2\uparrow$$

Với nước: K + H₂O → bazơ + H₂↑

$$K + H_2O \rightarrow \text{bazơ} + H_2\uparrow$$

Ví dụ: $$2Na + 2H_2O \rightarrow 2NaOH + H_2\uparrow$$

Với phi kim: K + phi kim → hợp chất

Ví dụ: $$2Fe + 3Cl_2 \rightarrow 2FeCl_3$$

2. Phi kim (P)

Đặc điểm chung:

  • Có tính oxi hóa
  • Nhận electron trong phản ứng hóa học
  • Tác dụng với kim loại, H₂, nước

Phản ứng điển hình:

Với kim loại: P + K → hợp chất

Với H₂: P + H₂ → hợp chất khí

Với nước: P + H₂O → axit

Chương 2: Oxit, Bazơ, Axit, Muối
1. Oxit (O)

Phân loại:

  • Oxit cơ bản: Oxit của kim loại (CuO, Fe₂O₃, CaO)
  • Oxit axit: Oxit của phi kim (CO₂, SO₂, SO₃)
  • Oxit lưỡng tính: Oxit của kim loại có tính lưỡng tính (Al₂O₃, ZnO)

Phản ứng:

Oxit cơ bản + axit → muối + nước

$$CuO + 2HCl \rightarrow CuCl_2 + H_2O$$

Oxit axit + bazơ → muối + nước

$$CO_2 + 2NaOH \rightarrow Na_2CO_3 + H_2O$$

Oxit lưỡng tính + axit/bazơ → muối + nước

$$Al_2O_3 + 6HCl \rightarrow 2AlCl_3 + 3H_2O$$ $$Al_2O_3 + 2NaOH \rightarrow 2NaAlO_2 + H_2O$$
2. Bazơ (B)

Đặc điểm:

  • Có vị chua, làm đổi màu quỳ tím thành xanh
  • Tác dụng với axit, oxit axit, muối

Phản ứng:

Bazơ + axit → muối + nước (phản ứng trung hòa)

$$NaOH + HCl \rightarrow NaCl + H_2O$$

Bazơ + oxit axit → muối + nước

$$2NaOH + CO_2 \rightarrow Na_2CO_3 + H_2O$$

Bazơ + muối → bazơ mới + muối mới

$$2NaOH + CuSO_4 \rightarrow Cu(OH)_2\downarrow + Na_2SO_4$$
3. Axit (A)

Đặc điểm:

  • Có vị chua, làm đổi màu quỳ xanh thành đỏ
  • Tác dụng với kim loại, bazơ, oxit cơ bản, muối

Phản ứng:

Axit + kim loại → muối + H₂↑

$$2HCl + Zn \rightarrow ZnCl_2 + H_2\uparrow$$

Axit + bazơ → muối + nước (trung hòa)

$$HCl + NaOH \rightarrow NaCl + H_2O$$

Axit + oxit cơ bản → muối + nước

$$2HCl + CuO \rightarrow CuCl_2 + H_2O$$

Axit + muối → axit mới + muối mới

$$2HCl + Na_2CO_3 \rightarrow 2NaCl + H_2O + CO_2\uparrow$$
4. Muối (M)

Phân loại:

  • Muối trung hòa: NaCl, KNO₃
  • Muối axit: NaHSO₄, KHCO₃
  • Muối bazơ: Cu(OH)Cl, Fe(OH)₂Cl

Phản ứng:

Muối + kim loại → muối mới + kim loại mới

$$CuSO_4 + Zn \rightarrow ZnSO_4 + Cu$$

Muối + bazơ → muối mới + bazơ mới

$$CuSO_4 + 2NaOH \rightarrow Cu(OH)_2\downarrow + Na_2SO_4$$

Muối + axit → muối mới + axit mới

$$CaCO_3 + 2HCl \rightarrow CaCl_2 + H_2O + CO_2\uparrow$$
Chương 4: Điện phân
1. Khái niệm

Điện phân là quá trình biến đổi điện năng thành hóa năng, xảy ra ở điện cực khi cho dòng điện đi qua dung dịch điện phân hoặc chất điện phân nóng chảy.

2. Điện phân dung dịch muối

Điện phân NaCl (nước):

Cátốt: 2H₂O + 2e⁻ → H₂↑ + 2OH⁻

$$2H_2O + 2e^- \rightarrow H_2\uparrow + 2OH^-$$

Anốt: 2Cl⁻ → Cl₂↑ + 2e⁻

$$2Cl^- \rightarrow Cl_2\uparrow + 2e^-$$

Phản ứng tổng: 2NaCl + 2H₂O → 2NaOH + H₂↑ + Cl₂↑

$$2NaCl + 2H_2O \rightarrow 2NaOH + H_2\uparrow + Cl_2\uparrow$$

Điện phân CuSO₄ (nước):

Cátốt: Cu²⁺ + 2e⁻ → Cu

$$Cu^{2+} + 2e^- \rightarrow Cu$$

Anốt: 2H₂O → O₂↑ + 4H⁺ + 4e⁻

$$2H_2O \rightarrow O_2\uparrow + 4H^+ + 4e^-$$

Phản ứng tổng: 2CuSO₄ + 2H₂O → 2Cu + 2H₂SO₄ + O₂↑

$$2CuSO_4 + 2H_2O \rightarrow 2Cu + 2H_2SO_4 + O_2\uparrow$$
3. Điện phân dung dịch axit

Điện phân H₂SO₄ (nước):

Cátốt: 2H₂O + 2e⁻ → H₂↑ + 2OH⁻

$$2H_2O + 2e^- \rightarrow H_2\uparrow + 2OH^-$$

Anốt: 2H₂O → O₂↑ + 4H⁺ + 4e⁻

$$2H_2O \rightarrow O_2\uparrow + 4H^+ + 4e^-$$

Phản ứng tổng: 2H₂O → 2H₂↑ + O₂↑

$$2H_2O \rightarrow 2H_2\uparrow + O_2\uparrow$$
⚗️ II. Hóa học hữu cơ
Chương 1: Hydrocarbon
1. Alkan (CₙH₂ₙ₊₂)

Đặc điểm:

  • Liên kết đơn C-C
  • Bão hòa với H₂
  • Tính chất: cháy, tác dụng với halogen

Phản ứng điển hình:

Đốt cháy: CₙH₂ₙ₊₂ + (3n+1)/2 O₂ → nCO₂ + (n+1)H₂O

$$C_nH_{2n+2} + \frac{3n+1}{2}O_2 \rightarrow nCO_2 + (n+1)H_2O$$

Tác dụng với halogen: CₙH₂ₙ₊₂ + X₂ → CₙH₂ₙ₊₁X + HX

$$C_nH_{2n+2} + X_2 \rightarrow C_nH_{2n+1}X + HX$$

Ví dụ:

  • CH₄ (metan)
  • C₂H₆ (etan)
  • C₃H₈ (propan)
2. Alken (CₙH₂ₙ)

Đặc điểm:

  • Có một liên kết đôi C=C
  • Không bão hòa với H₂
  • Tính chất: cháy, tác dụng với H₂, X₂, HX, H₂O

Phản ứng điển hình:

Đốt cháy: CₙH₂ₙ + 3n/2 O₂ → nCO₂ + nH₂O

$$C_nH_{2n} + \frac{3n}{2}O_2 \rightarrow nCO_2 + nH_2O$$

Tác dụng với H₂: CₙH₂ₙ + H₂ → CₙH₂ₙ₊₂

$$C_nH_{2n} + H_2 \rightarrow C_nH_{2n+2}$$

Tác dụng với X₂: CₙH₂ₙ + X₂ → CₙH₂ₙX₂

$$C_nH_{2n} + X_2 \rightarrow C_nH_{2n}X_2$$

Tác dụng với HX: CₙH₂ₙ + HX → CₙH₂ₙ₊₁X

$$C_nH_{2n} + HX \rightarrow C_nH_{2n+1}X$$

Ví dụ:

  • C₂H₄ (eten)
  • C₃H₆ (propen)
3. Alkin (CₙH₂ₙ₋₂)

Đặc điểm:

  • Có một liên kết ba C≡C
  • Không bão hòa với H₂
  • Tính chất: cháy, tác dụng với H₂, X₂, HX, H₂O

Phản ứng điển hình:

Đốt cháy: CₙH₂ₙ₋₂ + (3n-1)/2 O₂ → nCO₂ + (n-1)H₂O

$$C_nH_{2n-2} + \frac{3n-1}{2}O_2 \rightarrow nCO_2 + (n-1)H_2O$$

Tác dụng với H₂: CₙH₂ₙ₋₂ + 2H₂ → CₙH₂ₙ₊₂

$$C_nH_{2n-2} + 2H_2 \rightarrow C_nH_{2n+2}$$

Tác dụng với X₂: CₙH₂ₙ₋₂ + 2X₂ → CₙH₂ₙX₄

$$C_nH_{2n-2} + 2X_2 \rightarrow C_nH_{2n}X_4$$

Tác dụng với HX: CₙH₂ₙ₋₂ + 2HX → CₙH₂ₙX₂

$$C_nH_{2n-2} + 2HX \rightarrow C_nH_{2n}X_2$$

Ví dụ:

  • C₂H₂ (etin)
  • C₃H₄ (propin)
Chương 2: Các dẫn xuất hydrocarbon
1. Rượu (R-OH)

Đặc điểm:

  • Chứa nhóm -OH
  • Tính chất: cháy, tác dụng với kim loại, axit

Phản ứng điển hình:

Đốt cháy: C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O

$$C_2H_5OH + 3O_2 \rightarrow 2CO_2 + 3H_2O$$

Tác dụng với Na: 2C₂H₅OH + 2Na → 2C₂H₅ONa + H₂↑

$$2C_2H_5OH + 2Na \rightarrow 2C_2H_5ONa + H_2\uparrow$$

Tác dụng với axit: C₂H₅OH + CH₃COOH → CH₃COOC₂H₅ + H₂O

$$C_2H_5OH + CH_3COOH \rightarrow CH_3COOC_2H_5 + H_2O$$

Ví dụ:

  • CH₃OH (metanol)
  • C₂H₅OH (etanol)
2. Phenol (C₆H₅-OH)

Đặc điểm:

  • Nhóm -OH gắn trực tiếp vào vòng benzen
  • Tính chất: cháy, tác dụng với NaOH, FeCl₃

Phản ứng điển hình:

Tác dụng với NaOH: C₆H₅OH + NaOH → C₆H₅ONa + H₂O

$$C_6H_5OH + NaOH \rightarrow C_6H_5ONa + H_2O$$

Tác dụng với FeCl₃: C₆H₅OH + FeCl₃ → [C₆H₅O]₃Fe + 3HCl

$$C_6H_5OH + FeCl_3 \rightarrow [C_6H_5O]_3Fe + 3HCl$$
3. Aldehyde (R-CHO)

Đặc điểm:

  • Chứa nhóm -CHO
  • Tính chất: cháy, tác dụng với chất oxi hóa nhẹ

Phản ứng điển hình:

Đốt cháy: CH₃CHO + 5/2 O₂ → 2CO₂ + 2H₂O

$$CH_3CHO + \frac{5}{2}O_2 \rightarrow 2CO_2 + 2H_2O$$

Tác dụng với AgNO₃/NH₃: CH₃CHO + 2Ag(NH₃)₂OH → 2Ag↓ + CH₃COONH₄ + 3NH₃ + H₂O

$$CH_3CHO + 2Ag(NH_3)_2OH \rightarrow 2Ag\downarrow + CH_3COONH_4 + 3NH_3 + H_2O$$

Tác dụng với Cu(OH)₂: CH₃CHO + 2Cu(OH)₂ → CH₃COOH + Cu₂O↓ + 2H₂O

$$CH_3CHO + 2Cu(OH)_2 \rightarrow CH_3COOH + Cu_2O\downarrow + 2H_2O$$

Ví dụ:

  • HCHO (formaldehyde)
  • CH₃CHO (acetaldehyde)
4. Carboxylic acid (R-COOH)

Đặc điểm:

  • Chứa nhóm -COOH
  • Tính chất: cháy, tác dụng với rượu, kim loại

Phản ứng điển hình:

Đốt cháy: CH₃COOH + 2O₂ → 2CO₂ + 2H₂O

$$CH_3COOH + 2O_2 \rightarrow 2CO_2 + 2H_2O$$

Tác dụng với rượu: CH₃COOH + C₂H₅OH → CH₃COOC₂H₅ + H₂O

$$CH_3COOH + C_2H_5OH \rightarrow CH_3COOC_2H_5 + H_2O$$

Tác dụng với Na: 2CH₃COOH + 2Na → 2CH₃COONa + H₂↑

$$2CH_3COOH + 2Na \rightarrow 2CH_3COONa + H_2\uparrow$$

Ví dụ:

  • HCOOH (axit formic)
  • CH₃COOH (axit acetic)
5. Ester (R-COO-R')

Đặc điểm:

  • Chứa nhóm -COO-
  • Tính chất: cháy, tác dụng với nước

Phản ứng điển hình:

Đốt cháy: CH₃COOC₂H₅ + 5O₂ → 4CO₂ + 4H₂O

$$CH_3COOC_2H_5 + 5O_2 \rightarrow 4CO_2 + 4H_2O$$

Tác dụng với nước: CH₃COOC₂H₅ + H₂O → CH₃COOH + C₂H₅OH

$$CH_3COOC_2H_5 + H_2O \rightarrow CH_3COOH + C_2H_5OH$$

Ví dụ:

  • CH₃COOCH₃ (methyl acetate)
  • CH₃COOC₂H₅ (ethyl acetate)
🔬 III. Hóa học phân tích
Chương 1: Phản ứng nhận biết
1. Nhận biết ion

Ion NH₄⁺:

$$NH_4^+ + NaOH \rightarrow NH_3\uparrow + H_2O + Na^+$$

NH₃ làm ẩm quỳ đỏ chuyển thành xanh

Ion Cl⁻:

$$Cl^- + AgNO_3 \rightarrow AgCl\downarrow (\text{trắng}) + NO_3^-$$

AgCl không tan trong HNO₃

Ion SO₄²⁻:

$$SO_4^{2-} + BaCl_2 \rightarrow BaSO_4\downarrow (\text{trắng}) + 2Cl^-$$

BaSO₄ không tan trong HCl

Ion CO₃²⁻:

$$CO_3^{2-} + 2HCl \rightarrow CO_2\uparrow + H_2O + 2Cl^-$$

CO₂ làm vôi đục (Ca(OH)₂)

2. Nhận biết chất

Axit:

  • Làm quỳ xanh chuyển thành đỏ
  • Tác dụng với kim loại tạo H₂

Bazơ:

  • Làm quỳ đỏ chuyển thành xanh
  • Tác dụng với axit tạo muối + nước

Muối:

  • Tác dụng với axit/bazơ tạo khí
  • Tác dụng với AgNO₃/BaCl₂ tạo kết tủa
Chương 2: Phương pháp phân tích định tính
1. Phương pháp nhận biết
  • Quỳ: Dùng để nhận biết axit/bazơ
  • AgNO₃: Dùng để nhận biết ion Cl⁻, Br⁻, I⁻
  • BaCl₂: Dùng để nhận biết ion SO₄²⁻
  • NaOH: Dùng để nhận biết ion NH₄⁺
2. Phương pháp tách chất
  • Lọc: Tách chất rắn khỏi dung dịch
  • Chưng cất: Tách chất lỏng có nhiệt độ sôi khác nhau
  • Kết tinh: Tách chất rắn từ dung dịch bão hòa
  • Chiết xuất: Tách chất bằng dung môi khác
Chương 3: Phương pháp phân tích định lượng
1. Định lượng bằng thể tích
  • Định lượng axit/bazơ: Dùng dung dịch chuẩn để định lượng
  • Định lượng khí: Dùng thể tích khí để tính số mol
2. Định lượng bằng khối lượng
  • Định lượng chất rắn: Dùng khối lượng để tính số mol
  • Định lượng kết tủa: Dùng khối lượng kết tủa để tính số mol
3. Công thức tính

Số mol: n = m/M = V/V_m = N/N_A

Nồng độ mol: C = n/V

Nồng độ khối lượng: ρ = m/V

Phần trăm khối lượng: % = (m_chất/m_hòa tan) × 100%

⚡ IV. Phản ứng oxi hóa khử
1. Khái niệm cơ bản
  • Oxi hóa: Chất mất electron, tăng số oxi hóa
  • Khử: Chất nhận electron, giảm số oxi hóa
  • Chất oxi hóa: Chất có tính oxi hóa, nhận electron
  • Chất khử: Chất có tính khử, cho electron
2. Phản ứng oxi hóa khử điển hình
Phản ứng đốt cháy:
$$C + O_2 \rightarrow CO_2$$
$$S + O_2 \rightarrow SO_2$$
$$2Mg + O_2 \rightarrow 2MgO$$
Phản ứng thế:
$$Zn + CuSO_4 \rightarrow ZnSO_4 + Cu$$
$$Fe + CuSO_4 \rightarrow FeSO_4 + Cu$$
Phản ứng hợp thành:
$$2H_2 + O_2 \rightarrow 2H_2O$$
$$N_2 + 3H_2 \rightarrow 2NH_3$$
3. Tính khử của các halogen (I₂, Br₂, Cl₂)
⚠ Lưu ý: Các halogen (Cl₂, Br₂, I₂) có tính oxi hóa mạnh, nhưng cũng có thể thể hiện tính khử trong một số phản ứng đặc biệt với các chất oxi hóa mạnh hơn.
Iod (I₂) - Tính khử:
a) Tác dụng với H₂S (sulfua hidrat)
$$I_2 + H_2S \rightarrow S\downarrow + 2HI$$
Hiện tượng: Xuất hiện kết tủa màu vàng của lưu huỳnh
b) Tác dụng với SO₂ (lưu huỳnh điôxit)
$$I_2 + SO_2 + 2H_2O \rightarrow H_2SO_4 + 2HI$$
I₂ bị khử thành I⁻, SO₂ bị oxi hóa thành SO₄²⁻
c) Tác dụng với Na₂S₂O₃ (natri thiosunfat)
$$I_2 + 2Na_2S_2O_3 \rightarrow 2NaI + Na_2S_4O_6$$
Phản ứng dùng để định lượng I₂
d) Tác dụng với H₂O₂ (nước oxi hóa)
$$I_2 + H_2O_2 \rightarrow 2HIO$$
I₂ bị oxi hóa thành HIO
e) Tác dụng với Cl₂ (clo)
$$I_2 + 5Cl_2 + 6H_2O \rightarrow 2HIO_3 + 10HCl$$
I₂ bị oxi hóa mạnh thành HIO₃
f) Tác dụng với HNO₃ đặc (axit nitric đặc)
$$I_2 + 10HNO_3 \rightarrow 2HIO_3 + 10NO_2\uparrow + 4H_2O$$
I₂ bị oxi hóa thành HIO₃
Brom (Br₂) - Tính khử:
a) Tác dụng với H₂S (sulfua hidrat)
$$Br_2 + H_2S \rightarrow S\downarrow + 2HBr$$
Hiện tượng: Xuất hiện kết tủa màu vàng của lưu huỳnh
b) Tác dụng với SO₂ (lưu huỳnh điôxit)
$$Br_2 + SO_2 + 2H_2O \rightarrow H_2SO_4 + 2HBr$$
Br₂ bị khử thành Br⁻, SO₂ bị oxi hóa thành SO₄²⁻
c) Tác dụng với Na₂S₂O₃ (natri thiosunfat)
$$Br_2 + 2Na_2S_2O_3 \rightarrow 2NaBr + Na_2S_4O_6$$
Phản ứng dùng để định lượng Br₂
d) Tác dụng với H₂O₂ (nước oxi hóa)
$$Br_2 + H_2O_2 \rightarrow 2HBrO$$
Br₂ bị oxi hóa thành HBrO
e) Tác dụng với Cl₂ (clo)
$$Br_2 + 5Cl_2 + 6H_2O \rightarrow 2HBrO_3 + 10HCl$$
Br₂ bị oxi hóa mạnh thành HBrO₃
f) Tác dụng với HNO₃ đặc (axit nitric đặc)
$$Br_2 + 10HNO_3 \rightarrow 2HBrO_3 + 10NO_2\uparrow + 4H_2O$$
Br₂ bị oxi hóa thành HBrO₃
Clo (Cl₂) - Tính khử:
a) Tác dụng với H₂S (sulfua hidrat)
$$Cl_2 + H_2S \rightarrow S\downarrow + 2HCl$$
Hiện tượng: Xuất hiện kết tủa màu vàng của lưu huỳnh
b) Tác dụng với SO₂ (lưu huỳnh điôxit)
$$Cl_2 + SO_2 + 2H_2O \rightarrow H_2SO_4 + 2HCl$$
Cl₂ bị khử thành Cl⁻, SO₂ bị oxi hóa thành SO₄²⁻
c) Tác dụng với Na₂S₂O₃ (natri thiosunfat)
$$Cl_2 + 2Na_2S_2O_3 \rightarrow 2NaCl + Na_2S_4O_6$$
Phản ứng dùng để định lượng Cl₂
d) Tác dụng với H₂O₂ (nước oxi hóa)
$$Cl_2 + H_2O_2 \rightarrow 2HClO$$
Cl₂ bị oxi hóa thành HClO
e) Tác dụng với KMnO₄ (permanganat kali)
$$5Cl_2 + 2KMnO_4 + 8H_2O \rightarrow 2MnCl_2 + 2KCl + 8HClO$$
Cl₂ bị oxi hóa thành HClO
f) Tác dụng với HNO₃ đặc (axit nitric đặc)
$$Cl_2 + 10HNO_3 \rightarrow 2HClO_3 + 10NO_2\uparrow + 4H_2O$$
Cl₂ bị oxi hóa thành HClO₃
📊 Tóm tắt:
  • I₂ (Iod): Dễ bị oxi hóa nhất, có tính khử mạnh nhất trong nhóm halogen
  • Br₂ (Brom): Có tính khử trung bình
  • Cl₂ (Clo): Có tính khử yếu nhất, chủ yếu có tính oxi hóa
$$\text{Thứ tự tính khử tăng dần: } Cl_2 < Br_2 < I_2$$
🌡️ V. Halogen và các hợp chất
1. Khái niệm cơ bản

Halogen: Nhóm VIIA trong bảng tuần hoàn (F, Cl, Br, I)

Đặc điểm chung:

  • Phi kim điển hình
  • Có 7 electron lớp ngoài cùng
  • Số oxi hóa đặc trưng = -1 trong hợp chất
2. Tính chất của các halogen
Halogen Trạng thái Màu sắc Tính oxi hóa
Fluorine (F) Khí Vàng nhạt Mạnh nhất
Chlorine (Cl) Khí Vàng lục Rất mạnh
Bromine (Br) Lỏng Đỏ nâu Mạnh
Iodine (I) Rắn Tím đen Yếu hơn
📊 Thứ tự tính oxi hóa: F > Cl > Br > I
3. Phản ứng với kim loại
Phản ứng điển hình:

Fe + Cl₂ → FeCl₃

$$Fe + Cl_2 \rightarrow FeCl_3$$

Cu + Cl₂ → CuCl₂

$$Cu + Cl_2 \rightarrow CuCl_2$$

Al + Cl₂ → AlCl₃

$$2Al + 3Cl_2 \rightarrow 2AlCl_3$$

Zn + Cl₂ → ZnCl₂

$$Zn + Cl_2 \rightarrow ZnCl_2$$
4. Phản ứng với hydrogen
Phản ứng điển hình:

H₂ + F₂ → 2HF (phản ứng mãnh liệt, nổ)

$$H_2 + F_2 \rightarrow 2HF$$

H₂ + Cl₂ → 2HCl (phản ứng mạnh, cần ánh sáng hoặc nhiệt)

$$H_2 + Cl_2 \rightarrow 2HCl$$

H₂ + Br₂ → 2HBr (phản ứng ở nhiệt độ cao)

$$H_2 + Br_2 \rightarrow 2HBr$$

H₂ + I₂ ⇌ 2HI (phản ứng thuận nghịch)

$$H_2 + I_2 \rightleftharpoons 2HI$$
⚠️ Lưu ý: Phản ứng giữa I₂ và H₂ là phản ứng thuận nghịch, không diễn ra hoàn toàn.
5. Phản ứng thế
Phản ứng thế giữa các halogen:

Cl₂ + 2NaBr → 2NaCl + Br₂

$$Cl_2 + 2NaBr \rightarrow 2NaCl + Br_2$$

Cl₂ + 2NaI → 2NaCl + I₂

$$Cl_2 + 2NaI \rightarrow 2NaCl + I_2$$

Br₂ + 2NaI → 2NaBr + I₂

$$Br_2 + 2NaI \rightarrow 2NaBr + I_2$$
📊 Quy tắc: Halogen mạnh hơn sẽ thế halogen yếu hơn trong phản ứng thế.
6. Các hợp chất quan trọng
Hydrohalic acids:
  • HF: Acid yếu, ăn mòn thủy tinh hữu cơ
  • HCl, HBr, HI: Acid mạnh, tan tốt trong nước
Nước Javel:
  • Thành phần chính: NaClO (sodium hypochlorite)
  • Công thức sản xuất: Cl₂ + 2NaOH → NaCl + NaClO + H₂O
    • $$Cl_2 + 2NaOH \rightarrow NaCl + NaClO + H_2O$$
  • Tính chất: Tẩy màu, khử trùng
Nước muối sinh lý:
  • Nồng độ: 0,9% NaCl
  • Công dụng: Diệt khuẩn, sát trùng trong y học
7. Ứng dụng thực tế
Các ứng dụng quan trọng:
  • Chlorine: Khử trùng nước sinh hoạt, sản xuất nước Javel
  • Iodine: Tập trung ở tuyến giáp trong cơ thể người
  • Nước muối sinh lý: Diệt khuẩn, sát trùng trong y học
  • Nước Javel: Tẩy màu, khử trùng
8. Điều cần lưu ý
  • Fluorine chỉ có số oxi hóa -1 trong hợp chất
  • Chlorine vừa có tính oxi hóa vừa có tính khử
  • Iodine tập trung ở tuyến giáp trong cơ thể người
  • Chlorine dùng để khử trùng nước sinh hoạt
  • Trong phản ứng thế, halogen mạnh hơn sẽ thế halogen yếu hơn
⏱️ VI. Tốc độ phản ứng
1. Khái niệm cơ bản

Tốc độ phản ứng: Mức độ xảy ra nhanh hay chậm của phản ứng hóa học

Đơn vị: mol/(L·s) hoặc mol/(L·min)

2. Biểu thức tốc độ phản ứng
Phản ứng tổng quát: aA + bB → cC + dD

Biểu thức tốc độ: v = k·[A]ᵃ·[B]ᵇ

$$v = k \cdot [A]^a \cdot [B]^b$$

Trong đó:

  • v: Tốc độ phản ứng
  • k: Hằng số tốc độ
  • [A], [B]: Nồng độ các chất tham gia
  • a, b: Bậc phản ứng theo từng chất
Ví dụ cụ thể:

Phản ứng: X + 2Y → 2Z

$$v = k \cdot [X] \cdot [Y]^2$$

Phản ứng: 2NO + O₂ → 2NO₂

$$v = k \cdot [NO]^2 \cdot [O_2]$$

Phản ứng: 2X + Y → 2Z

$$v = k \cdot [X]^2 \cdot [Y]$$
3. Các yếu tố ảnh hưởng đến tốc độ phản ứng
1. Nồng độ chất tham gia phản ứng
  • Tăng nồng độ → Tăng tốc độ phản ứng
  • Tăng nồng độ lên n lần → Tốc độ tăng lên nᵐ lần (m = bậc phản ứng)
2. Nhiệt độ
  • Tăng nhiệt độ → Tăng tốc độ phản ứng
  • Quy tắc: Tăng 10°C → Tốc độ tăng 2-3 lần
3. Diện tích tiếp xúc
  • Tăng diện tích → Tăng tốc độ phản ứng
  • Phản ứng ở bề mặt diễn ra nhanh hơn
4. Chất xúc tác
  • Chất xúc tác tăng tốc độ nhưng không bị tiêu hao
  • Ví dụ: MnO₂ xúc tác phân giải H₂O₂
5. Áp suất (cho phản ứng khí)
  • Tăng áp suất → Tăng nồng độ → Tăng tốc độ
  • Áp suất ảnh hưởng đến phản ứng có khí
4. Phản ứng phân giải H₂O₂
Phản ứng:

2H₂O₂ → 2H₂O + O₂

$$2H_2O_2 \rightarrow 2H_2O + O_2$$
Chất xúc tác:
  • MnO₂ (bột đen)
  • Không bị tiêu hao sau phản ứng
Biểu thức tốc độ:

v = k·[H₂O₂]²·[O₂]

$$v = k \cdot [H_2O_2]^2 \cdot [O_2]$$
5. Điều cần lưu ý
  • Tốc độ phản ứng không phụ thuộc vào thời gian xảy ra phản ứng
  • Theo thời gian, nồng độ chất tham gia giảm → Tốc độ phản ứng giảm
  • Chất xúc tác không bị tiêu hao sau phản ứng
  • Tăng nồng độ chất tham gia lên n lần → Tốc độ tăng lên nᵐ lần
6. Ví dụ bài toán
Ví dụ 1: Phản ứng X + 2Y → 2Z
  • k = 0,6 (mol⁻²·L²·s⁻¹)
  • [X]₀ = 0,5 M, [Y]₀ = 0,8 M
  • Khi [Y] giảm còn 0,2 M

v = 0,6 × 0,5 × (0,2)² = 0,012 mol/(L·s)

$$v = 0.6 \times 0.5 \times (0.2)^2 = 0.012 \text{ mol/(L·s)}$$
Ví dụ 2: Phản ứng 2X + Y → 2Z
  • k = 0,8 (mol⁻²·L²·s⁻¹)
  • [X]₀ = 0,6 M, [Y]₀ = 0,9 M
  • Khi [Y] giảm 0,1 M → [Y] = 0,8 M

v = 0,8 × (0,6)² × 0,8 = 0,2304 mol/(L·s)

$$v = 0.8 \times (0.6)^2 \times 0.8 = 0.2304 \text{ mol/(L·s)}$$
🔥 VII. Nhiệt hóa học
1. Khái niệm cơ bản

Nhiệt phản ứng (ΔH): Nhiệt lượng tỏa ra hoặc thu vào trong phản ứng

Đơn vị: kJ/mol hoặc kJ

2. Phân loại phản ứng theo nhiệt
Phản ứng tỏa nhiệt (ΔH < 0):
  • Giải phóng nhiệt lượng ra môi trường
  • Ví dụ: 4NH₃ + 3O₂ → 2N₂ + 6H₂O (ΔH = -1531 kJ)
    • $$4NH_3 + 3O_2 \rightarrow 2N_2 + 6H_2O \quad (\Delta H = -1531 \text{ kJ})$$
Phản ứng thu nhiệt (ΔH > 0):
  • Hấp thu nhiệt lượng từ môi trường
  • Ví dụ: 2NaHCO₃ → Na₂CO₃ + H₂O + CO₂ (ΔH = +20,33 kJ)
    • $$2NaHCO_3 \rightarrow Na_2CO_3 + H_2O + CO_2 \quad (\Delta H = +20.33 \text{ kJ})$$
📊 Quy tắc:
  • ΔH < 0: Phản ứng tỏa nhiệt
  • ΔH > 0: Phản ứng thu nhiệt
3. Nhiệt tạo thành chuẩn (ΔfH°)
Định nghĩa:

Nhiệt lượng khi tạo thành 1 mol chất từ các nguyên tố ở trạng thái chuẩn

Đơn vị: kJ/mol

Đặc điểm:
  • ΔfH° của đơn chất = 0
  • ΔfH° của hợp chất thường ≠ 0
Ví dụ:
Chất ΔfH° Giải thích
H₂O(l) ≠ 0 Hợp chất
N₂(g) = 0 Đơn chất
CH₄(g) ≠ 0 Hợp chất
CuO(s) ≠ 0 Hợp chất
O₂(g) = 0 Đơn chất
4. Phương trình nhiệt hóa học
Phản ứng hợp thành ethylen:

C₂H₄(g) + H₂(g) → C₂H₆(g) ΔH = -137,0 kJ

$$C_2H_4(g) + H_2(g) \rightarrow C_2H_6(g) \quad \Delta H = -137.0 \text{ kJ}$$

→ Giải phóng 137,0 kJ nhiệt

Phản ứng phân giải CO₂:

CO₂(g) → CO(g) + O₂(g) ΔH = +280,0 kJ

$$CO_2(g) \rightarrow CO(g) + O_2(g) \quad \Delta H = +280.0 \text{ kJ}$$

→ Hấp thu 280,0 kJ nhiệt

Phản ứng phân giải NaHCO₃:

2NaHCO₃(s) → Na₂CO₃(s) + H₂O(l) + CO₂(g) ΔH = +20,33 kJ

$$2NaHCO_3(s) \rightarrow Na_2CO_3(s) + H_2O(l) + CO_2(g) \quad \Delta H = +20.33 \text{ kJ}$$

→ Hấp thu 20,33 kJ nhiệt

Phản ứng oxi hóa NH₃:

4NH₃(g) + 3O₂(g) → 2N₂(g) + 6H₂O(l) ΔH = -1531 kJ

$$4NH_3(g) + 3O_2(g) \rightarrow 2N_2(g) + 6H_2O(l) \quad \Delta H = -1531 \text{ kJ}$$

→ Giải phóng 1531 kJ nhiệt

5. Điều cần lưu ý
  • ΔH < 0: Phản ứng tỏa nhiệt
  • ΔH > 0: Phản ứng thu nhiệt
  • Đơn vị enthalpy: kJ/mol
  • Đơn vị nhiệt phản ứng: kJ
  • Nhiệt tạo thành chuẩn của đơn chất luôn bằng 0
6. Ví dụ bài toán
Ví dụ 1: Xác định loại phản ứng
  • C₂H₄(g) + H₂(g) → C₂H₆(g) ΔH = -137,0 kJ
  • → Phản ứng tỏa nhiệt (giải phóng 137,0 kJ)
Ví dụ 2: Xác định chất có ΔfH° = 0
  • H₂O(l): ΔfH° ≠ 0 (hợp chất)
  • N₂(g): ΔfH° = 0 (đơn chất)
  • CH₄(g): ΔfH° ≠ 0 (hợp chất)
  • O₂(g): ΔfH° = 0 (đơn chất)
4. Hệ số nhiệt độ (γ)
1. Công thức tính hệ số nhiệt độ γ

Nếu bạn biết tốc độ phản ứng hoặc hằng số tốc độ ở hai nhiệt độ khác nhau, bạn có thể tính γ bằng công thức:

Công thức:

$$\gamma = \left( \frac{v_2}{v_1} \right)^{\frac{10}{T_2 - T_1}} \quad \text{hoặc} \quad \gamma = \left( \frac{k_2}{k_1} \right)^{\frac{10}{T_2 - T_1}}$$

Trong đó:

  • $T_1, T_2$: Nhiệt độ tại hai thời điểm khảo sát (có thể dùng đơn vị $^\circ C$ hoặc $K$)
  • $v_1, v_2$: Tốc độ phản ứng tương ứng với nhiệt độ $T_1$ và $T_2$
  • $k_1, k_2$: Hằng số tốc độ phản ứng tương ứng với nhiệt độ $T_1$ và $T_2$
2. Công thức tính sự thay đổi tốc độ phản ứng

Đây là dạng bài tập phổ biến nhất. Khi biết hệ số γ và sự thay đổi nhiệt độ, bạn có thể tìm xem tốc độ phản ứng tăng lên bao nhiêu lần ($n$):

Công thức:

$$n = \frac{v_2}{v_1} = \gamma^{\frac{T_2 - T_1}{10}}$$

Ví dụ nhanh:

Nếu một phản ứng có $\gamma = 2$, khi tăng nhiệt độ từ $20^\circ C$ lên $50^\circ C$, tốc độ phản ứng sẽ tăng:

$$n = 2^{\frac{50-20}{10}} = 2^3 = 8 \text{ (lần)}$$
3. Mối liên hệ với thời gian phản ứng

Vì tốc độ phản ứng ($v$) tỉ lệ nghịch với thời gian phản ứng ($t$) để cùng tiêu thụ một lượng chất như nhau, ta có:

Công thức:

$$\frac{v_2}{v_1} = \frac{t_1}{t_2} = \gamma^{\frac{T_2 - T_1}{10}}$$
⚠️ Lưu ý: Nếu đề bài cho thời gian, hãy cẩn thận vì nhiệt độ tăng thì thời gian phản ứng phải giảm xuống.
4. Ví dụ bài toán

Ví dụ 1: Tính hệ số nhiệt độ

  • Tại $20^\circ C$, tốc độ phản ứng là $v_1 = 0.5$ mol/(L·s)
  • Tại $40^\circ C$, tốc độ phản ứng là $v_2 = 2.0$ mol/(L·s)

Tính γ:

$$\gamma = \left( \frac{2.0}{0.5} \right)^{\frac{10}{40-20}} = 4^{\frac{10}{20}} = 4^{0.5} = 2$$

Ví dụ 2: Tính tốc độ phản ứng mới

  • Hệ số nhiệt độ $\gamma = 3$
  • Tốc độ phản ứng tại $25^\circ C$ là $v_1 = 0.6$ mol/(L·s)
  • Tính tốc độ phản ứng tại $55^\circ C$

Tính v₂:

$$n = 3^{\frac{55-25}{10}} = 3^3 = 27$$ $$v_2 = 27 \times 0.6 = 16.2 \text{ mol/(L·s)}$$

Ví dụ 3: Tính thời gian phản ứng

  • Hệ số nhiệt độ $\gamma = 2$
  • Thời gian phản ứng tại $20^\circ C$ là $t_1 = 60$ phút
  • Tính thời gian phản ứng tại $40^\circ C$

Tính t₂:

$$\frac{t_1}{t_2} = 2^{\frac{40-20}{10}} = 2^2 = 4$$ $$t_2 = \frac{60}{4} = 15 \text{ phút}$$
🧮 VIII. Công thức quan trọng
1. Công thức tính nồng độ

Nồng độ mol: $$C = \frac{n}{V_{dd}}$$

Trong đó: C (mol/L), n (số mol), $V_{dd}$ (thể tích dung dịch - L)

Nồng độ khối lượng: $$\rho = \frac{m}{V}$$

Trong đó: $\rho$ (g/L), m (khối lượng - g), V (thể tích - L)

Phần trăm khối lượng: $$\% = \frac{m_{\text{chất}}}{m_{\text{hòa tan}}} \times 100\%$$

Trong đó: $m_{\text{chất}}$ (khối lượng chất - g), $m_{\text{hòa tan}}$ (khối lượng dung dịch - g)

2. Công thức tính số mol

Từ khối lượng: $$n = \frac{m}{M}$$

Trong đó: n (số mol), m (khối lượng - g), M (khối lượng mol - g/mol)

Từ thể tích khí (đktc): $$n = \frac{V_{khí}}{V_m} = \frac{V_{khí}}{22.4}$$

Trong đó: n (số mol), $V_{khí}$ (thể tích khí - L), $V_m = 22.4$ L/mol (đktc)

Từ số phân tử: $$n = \frac{N}{N_A}$$

Trong đó: n (số mol), N (số phân tử), $N_A = 6.022 \times 10^{23}$ mol⁻¹

Từ nồng độ mol: $$n = C \times V_{dd}$$

Trong đó: n (số mol), C (nồng độ mol - mol/L), $V_{dd}$ (thể tích dung dịch - L)

3. Công thức tính khối lượng mol

Khối lượng mol: $$M = \frac{m}{n}$$

Trong đó: M (g/mol), m (khối lượng - g), n (số mol)

Từ công thức hóa học: $$M = \sum_{i=1}^{n} A_i \times n_i$$

Trong đó: $A_i$ (khối lượng nguyên tử), $n_i$ (số nguyên tử)

4. Công thức tính thể tích

Thể tích khí (đktc): $$V_{khí} = n \times V_m = n \times 22.4$$

Trong đó: $V_{khí}$ (L), n (số mol), $V_m = 22.4$ L/mol

Thể tích dung dịch: $$V_{dd} = \frac{n}{C}$$

Trong đó: $V_{dd}$ (L), n (số mol), C (nồng độ mol - mol/L)

5. Công thức tính khối lượng

Từ số mol: $$m = n \times M$$

Trong đó: m (g), n (số mol), M (g/mol)

Từ nồng độ: $$m = C \times V_{dd} \times M$$

Trong đó: m (g), C (mol/L), $V_{dd}$ (L), M (g/mol)

Từ phần trăm: $$m_{\text{chất}} = \frac{\% \times m_{\text{hòa tan}}}{100}$$

Trong đó: $m_{\text{chất}}$ (g), % (phần trăm), $m_{\text{hòa tan}}$ (g)

6. Công thức hiệu suất

Hiệu suất: $$\eta = \frac{m_{\text{thực}}}{m_{\text{lý thuyết}}} \times 100\%$$

Trong đó: $\eta$ (%), $m_{\text{thực}}$ (khối lượng thực tế - g), $m_{\text{lý thuyết}}$ (khối lượng lý thuyết - g)

Khối lượng thực tế: $$m_{\text{thực}} = \frac{\eta \times m_{\text{lý thuyết}}}{100}$$

Trong đó: $m_{\text{thực}}$ (g), $\eta$ (%), $m_{\text{lý thuyết}}$ (g)

7. Công thức phản ứng hóa học

Định luật bảo toàn khối lượng: $$\sum m_{\text{phản ứng}} = \sum m_{\text{sản phẩm}}$$

Định luật bảo toàn số nguyên tử: $$\sum n_{\text{phản ứng}} = \sum n_{\text{sản phẩm}}$$

Tỷ lệ mol: $$\frac{n_A}{n_B} = \frac{a}{b} \quad \text{(theo phương trình: } aA + bB \rightarrow cC + dD)$$

1. Axit mạnh
  • HCl, HNO₃, H₂SO₄
  • Điện li hoàn toàn trong nước
2. Bazơ mạnh
  • NaOH, KOH, Ca(OH)₂
  • Điện li hoàn toàn trong nước
3. Muối quan trọng
  • NaCl, KNO₃, CaCO₃
  • Các muối tan và không tan
4. Oxit quan trọng

5. Bài toán làm muối
Công thức cơ bản:

m_muối = V_nước × C_NaCl × η

$$m_{\text{muối}} = V_{\text{nước}} \times C_{\text{NaCl}} \times \eta$$

Trong đó:

  • V_nước: Thể tích nước biển (L)
  • C_NaCl: Nồng độ NaCl trong nước biển (g/L)
  • η: Hiệu suất quá trình (%)
Ví dụ 1:
  • V = 500.000 L
  • C = 30 g/L
  • η = 70%

m = 500.000 × 30 × 0,7 = 10.500.000 g = 10,5 tấn

$$m = 500.000 \times 30 \times 0.7 = 10.500.000 \text{ g} = 10.5 \text{ tấn}$$
Ví dụ 2:
  • V = 600.000 L
  • C = 35 g/L
  • η = 60%

m = 600.000 × 35 × 0,6 = 12.600.000 g = 12,6 tấn

$$m = 600.000 \times 35 \times 0.6 = 12.600.000 \text{ g} = 12.6 \text{ tấn}$$
6. Bài toán công nghiệp sản xuất chlorine và nước Javel
Phản ứng điện phân NaCl:

2NaCl + 2H₂O → 2NaOH + H₂↑ + Cl₂↑

$$2NaCl + 2H_2O \rightarrow 2NaOH + H_2\uparrow + Cl_2\uparrow$$
Phản ứng sản xuất nước Javel:

Cl₂ + 2NaOH → NaCl + NaClO + H₂O

$$Cl_2 + 2NaOH \rightarrow NaCl + NaClO + H_2O$$
Công thức tính:

m_NaClO = m_Cl₂ × %_sử_dụng × η × (M_NaClO/M_Cl₂)

$$m_{\text{NaClO}} = m_{\text{Cl}_2} \times \%\_{\text{sử dụng}} \times \eta \times \left(\frac{M_{\text{NaClO}}}{M_{\text{Cl}_2}}\right)$$

Trong đó:

  • m_Cl₂: Khối lượng chlorine sản xuất
  • %_sử_dụng: Tỷ lệ chlorine dùng làm nước Javel
  • η: Hiệu suất phản ứng
  • M_NaClO = 74,5 g/mol
  • M_Cl₂ = 71 g/mol
Ví dụ 1:
  • m_Cl₂ = 710 kg/ngày
  • %_sử_dụng = 20%
  • η = 100%
  • Nước Javel chứa 10% NaClO

m_NaClO = 710 × 0,2 × 1,0 × (74,5/71) = 149 kg

m_nước_Javel = 149/0,1 = 1490 kg = 1,49 tấn

$$m_{\text{NaClO}} = 710 \times 0.2 \times 1.0 \times \left(\frac{74.5}{71}\right) = 149 \text{ kg}$$ $$m_{\text{nước Javel}} = \frac{149}{0.1} = 1490 \text{ kg} = 1.49 \text{ tấn}$$
Ví dụ 2:
  • m_Cl₂ = 3,55 tấn/ngày
  • %_sử_dụng = 30%
  • η = 80%

m_NaClO = 3550 × 0,3 × 0,8 × (74,5/71) = 896 kg = 0,896 tấn

$$m_{\text{NaClO}} = 3550 \times 0.3 \times 0.8 \times \left(\frac{74.5}{71}\right) = 896 \text{ kg} = 0.896 \text{ tấn}$$
7. Bài toán tốc độ phản ứng
Công thức cơ bản:

v = k·[A]ᵃ·[B]ᵇ

$$v = k \cdot [A]^a \cdot [B]^b$$
Ví dụ 1: Phản ứng X + 2Y → 2Z
  • k = 0,6 (mol⁻²·L²·s⁻¹)
  • [X]₀ = 0,5 M, [Y]₀ = 0,8 M
  • Khi [Y] giảm còn 0,2 M

v = 0,6 × 0,5 × (0,2)² = 0,012 mol/(L·s)

$$v = 0.6 \times 0.5 \times (0.2)^2 = 0.012 \text{ mol/(L·s)}$$
Ví dụ 2: Phản ứng 2X + Y → 2Z
  • k = 0,8 (mol⁻²·L²·s⁻¹)
  • [X]₀ = 0,6 M, [Y]₀ = 0,9 M
  • Khi [Y] giảm 0,1 M → [Y] = 0,8 M

v = 0,8 × (0,6)² × 0,8 = 0,2304 mol/(L·s)

$$v = 0.8 \times (0.6)^2 \times 0.8 = 0.2304 \text{ mol/(L·s)}$$
8. Điều cần lưu ý khi làm bài toán
  • Chuyển đổi đơn vị: 1 tấn = 1.000 kg = 1.000.000 g
  • Hiệu suất luôn < 100% trong thực tế
  • Khi tính tốc độ phản ứng, cần sử dụng nồng độ tại thời điểm đó
  • Trong phản ứng thế, halogen mạnh hơn sẽ thế halogen yếu hơn
  • Luôn kiểm tra đơn vị của kết quả
